# Shen + FastAPI Example

Even if not the most performant, Python's ecosystem is huge. FastAPI + Pydantic provides excellent DX, auto OpenAPI docs, and async support.

**Usefulness**:
- Rapid prototyping.
- Great for data-heavy apps (ML, analytics) where Python shines.
- Pydantic models can be generated or aligned with Shen specs.
- Easy to integrate with other Python tools.

**Contrarian perspective**: Don't dismiss FastAPI just because of "not performant". For many use cases, the bottleneck is not CPU but developer time or integration with data science libs. Use this to show that Shen makes any stack safer. Benchmark real-world scenarios.
